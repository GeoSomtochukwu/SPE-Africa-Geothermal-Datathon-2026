## Problem Statement
Assess the geothermal potential of the Rotliegend sandstone using well data and design a hybrid heating & cooling system to meet **≥10 MWth heating** and **≥5 MWth cooling** demand for an urban district in Utrecht.

## Key Results
- **Best Well**: BLT-01 (49.15 m net thickness, 15.91% porosity, 138.55 mD permeability)
- **Deliverability**: 180 m³/h flow rate → **7.22 MWth** base load
- **Recommendation**: Hybrid geothermal system (doublet + heat pumps + TES)

## Repository Structure
- `SPE2026DATATHON CODE.ipynb` — Main analysis notebook
- `plots/` — All generated visualizations (well logs, demand profiles, system diagram)
- `outputs/` — Excel summary tables and results
- `Raw/` — Input datasets (LAS files, Excel data)

## How to Reproduce Results
1. Clone or download this repository.
2. Place raw data files in the `Raw/` folder (if not already included).
3. Open `SPE2026DATATHON CODE.ipynb` in Jupyter Notebook / JupyterLab.
4. Run **"Restart & Run All"**.
5. All plots and Excel outputs will be generated automatically in `plots/` and `outputs/` folders.
